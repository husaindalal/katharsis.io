Wildfly

This example presents Katharsis integration of Wildfly application server and Weld CDI.

Katharsis is added by a runtime-discoverable KatharsisFeature and a sample repository uses LinkedHashSet to perform operations on resources.

There is no web.xml provided (which is possible since Java EE 7) so there is need for a dummy Application implemented in WildflyApplication.

The example is available here: https://github.com/katharsis-project/katharsis-framework/tree/master/katharsis-examples/wildfly-example
