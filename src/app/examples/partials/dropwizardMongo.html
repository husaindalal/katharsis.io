Dropwizard + MongoDB

Example which uses Dropwizard, dropwizard-guice extension for dependency injection and MongoDB database along with Morphia document mapper.

The repositories contain a reference to a Datastore which handles the object-document mapping.

The example is available here: https://github.com/katharsis-project/katharsis-framework/tree/master/katharsis-examples/dropwizard-mongo-example
