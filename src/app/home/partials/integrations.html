Standard frameworks integration

Katharsis can be easily integrated into server side frameworks.

Database integration

By implementing defined repositories and resources it is easy integrate Katharsis with:

  • SQL solutions like Hibernate or JOOQ
  • No-SQL products like MongoDB and other key-value storage
  • In-memory custom storage
  • And many more...

JAX-RS integration

Katharsis uses internally Jackson for JSON manipulation and can be also used in resource's definition.

JAX-RS integration can be applied in less than 5 minutes and it won't break current implementation.

CDI integration

Katharsis can automatically pickup repository, module, exception mapper, etc. implementations from CDI environments.

Servlet integration

Katharsis can be used with other frameworks which use Java Servlet API.

Spring integration

Katharsis provide an easy integration with Spring and Spring Boot. For Spring Boot it provides a configuration class. Katharsis can automatically pickup repository, module, exception mapper, etc. implementations from an ApplicationContext. There is further support for Spring Security and JPA.

JSR-303 Bean Validation integration

Katharsis provides exceptions mappers for JSR-303 validation exceptions, allowing their translation into fully JSON API compliant errors.

Brave/Zipkin integration

Katharsis provides a Brave integration to allow the tracing of repository called with the Zipkin protocol.

JPA integration

Katharsis provides a JPA integration with allows to expose existing JPA entities without any additional implementation work.
